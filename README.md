# Visor de Excel

Proyecto Next.js para leer el archivo `base/costo historico.xlsx` y mostrar la hoja `costo`.

## Como se trabaja

1. Mantener el archivo Excel dentro de la carpeta `base`.
2. La hoja que se muestra debe llamarse `costo`.
3. Subir el proyecto a GitHub.
4. Importar o redeployar el repositorio en Vercel.

## Comandos

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.
